# Traditional Data vs Big Data

## Traditional Data

- Small in size  
- Stored in relational databases  
- Mostly structured  
- Processed using SQL  
- Used in small business applications  

### Tools Used

- MySQL  
- Oracle  
- MS SQL Server  
- Excel  

## Big Data

- Extremely large datasets  
- Structured, semi-structured, and unstructured  
- Requires distributed systems  
- Needs parallel processing  
- Generated at high speed  

### Tools Used

- Hadoop  
- Spark  
- Hive  
- BigQuery  
- Azure Data Lake
